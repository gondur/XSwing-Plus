XSwing Plus v0.355

English:

Start XSwing Plus:
- For Windows users:
  Simply run it with the XSwing Plus.exe.
- For Mac users:
  Simply run the xswing_v3.5.jar.
  Make sure Java 1.6 is used in Java configuration.
- For Linux users:
  You should use the xswing.sh script to start XSwing Plus:
  chmod +x  xswing.sh
  xswing.sh
  Make sure your zip software also extracted the folders (no flat extraction).

Short instruction:
 The objective is to simultaneously disband as many heavy balls as possible. To do this, simply place three 
 balls with the same color in a horizontal row. Now these and all adjoining balls of this color will disappear.
 Five balls of the same color in a column will shrink to one heavy ball. Every 50 balls a new color will be added. 
 Pay attention: One stack can only take up to eight balls.

Keys:
 Left/Right: Move the crane
 Down:       Drop a ball
 N:          New Game
 B:          Change the ball set
 E:          Disable particle effects
 ESC:        Exit the game 
 F2:         Toggle Fullscreen
 A/S/D:      Left/Down/Right Player 2

Pay attention:
 The game is in a very early development progress.
 Please inform me about bugs, suggestion or incitations to: XSwing@tobsefritz.de

- To run XSwing you nee at least Java JRE 6 Update 7 (The Java SE Runtime Environment), get it on: www.java.com/getjava/
- To start the game in window mode run the game with the parameter "true" or execute "XSwing_debug.bat".

XSwing Plus homepage: http://XSwing.net

Schnellanleitung:
 Ziel des Spiels ist es, möglichst viele schwere Kugeln auf einmal aufzulösen, indem du drei 
 gleichfarbige Kugeln in einer horizontalen Reihe nebeneinander platziert. Die drei und alle 
 angrenzenden Kugeln mit derselben Farbe lösen sich dann auf.
 Fünf gestapelte Kugeln schrumpfen zu einer schweren zusammen.
 Alle 50 Kugeln erreichst du ein höheres Level und eine neue Farbe kommt hinzu.
 Schnell hintereinander aufgelöste Reihen belohnt der Punkte Multiplikator mit einem Bonus.
 Doch aufgepasst: Jeder Stapel kann nur acht Bälle auf einmal tragen.

Tasten:
 Links/Rechts:  Kran bewegen
 Unten:         Kugel fallen lassen
 N:             Neues Spiel
 P:             Pause
 B:             Kugel Set wechseln
 E:             Partikel Effekte abschalten
 ESC:           Spiel beenden
 F2:            Vollbild umschalten
 A/S/D:         Links/Runter/Rechts Spieler 2
